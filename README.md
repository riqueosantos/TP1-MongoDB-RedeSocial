# Trabalho Prático 1 - Rede Social

`Análise e Desenvolvimento de Sistemas - PUC Minas`

`Banco de Dados NoSQL`

Aplicação web inspirada no X/Twitter, desenvolvida como Trabalho Prático 1 da disciplina de Banco de Dados NoSQL (ADS - PUC Minas, 1/2026).

A aplicação utiliza MongoDB como banco de dados e demonstra modelagem NoSQL com documentos aninhados, referências por ObjectId, operadores avançados e o Aggregation Framework.
